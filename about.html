🎸 About GuitarFB

我们的使命

Our Mission

GuitarFB 是一个免费的在线吉他指板学习工具，旨在帮助吉他爱好者更好地理解和掌握指板上的音阶、和弦和音程关系。无论你是初学者还是进阶玩家，都能在这里找到适合自己的练习方式。

GuitarFB is a free online guitar fretboard learning tool designed to help guitar enthusiasts better understand and master scales, chords, and intervals on the fretboard. Whether you're a beginner or an advanced player, you'll find practice methods that suit your level.

核心功能

Key Features

  • 可视化指板 - 清晰展示吉他指板上的音符位置
  • 多种音阶 - 支持14种音阶类型
  • 和弦生成 - 自动生成三和弦到十三和弦
  • 自定义调弦 - 支持标准及多种特殊调弦
  • 节拍器 - 内置节拍器功能
  • 标注系统 - 添加自定义标注
  • 数据导出 - JSON格式导入导出
  • Visual Fretboard - Clear note position display
  • Multiple Scales - 14 scale types supported
  • Chord Generation - Triads through 13th chords
  • Custom Tuning - Standard and alternate tunings
  • Metronome - Built-in rhythm trainer
  • Annotation System - Custom markers on fretboard
  • Data Export - JSON import/export

使用说明

How to Use

1. 选择调性和音阶/和弦类型，点击生成按钮

2. 指板上会以彩色标注突出显示对应的音符位置

3. 点击指板可添加自定义标注，也可切换音高模式标注音符

4. 节拍器支持 BPM 调节、拍号选择、节奏细分和定时练习

5. 使用🎨按钮切换 8 套配色主题，选择适合你的视觉风格

1. Select a key and scale/chord type, click the generate button

2. The fretboard highlights corresponding note positions with color-coded markers

3. Tap the fretboard to add custom annotations, or switch to pitch mode for note labels

4. The metronome supports BPM, time signature, rhythm subdivision, and countdown timer

5. Use the 🎨 button to switch between 8 color palettes

适合人群

Who It's For

吉他初学者：可视化指板帮助你快速记忆音名位置和各调音阶指法，无需再依赖指板贴纸。配合和弦生成功能，理解三和弦、七和弦等常见和弦的构成逻辑。

进阶演奏者：利用多指板对比不同调式（如大调与混合利底亚的差异），通过音程标注模式理解指板上的音程关系，提升即兴演奏能力。

音乐教育者：可用作课堂教学辅助工具，为学生直观演示音阶与和弦在指板上的分布，支持中英文双语切换方便双语教学。

Beginners: The visual fretboard helps you quickly memorize note positions and scale patterns without relying on fretboard stickers. Chord generation helps you understand how triads, 7ths, and other chords are built.

Intermediate & Advanced Players: Compare different modes side by side using multiple fretboards. Use interval annotation mode to understand fretboard intervals and improve improvisation.

Educators: Use as a classroom teaching aid to visually demonstrate scales and chords. Bilingual Chinese/English support is useful for music instruction.

配色系统

Color Palettes

GuitarFB 内置 8 套精心设计的配色方案，涵盖从浅色到深色、从复古到赛博的不同视觉风格。每套方案包含 10 种相互协调的颜色，分别用于页面背景、卡片、面板、画布、文字、按钮、强调色等不同元素。通过 🎨 按钮随时切换，适应不同的使用环境和个人偏好。

GuitarFB features 8 carefully crafted color palettes, ranging from light to dark, from retro to cyber. Each palette includes 10 harmonized colors for background, card, panel, canvas, text, button, accent, and more. Switch anytime via the 🎨 button.

技术特点

Technical Highlights

• 完全在浏览器中运行，无需安装任何软件，所有数据处理均在本地完成
• 支持 PC 和移动设备，触屏操作友好
• 响应式设计，适配各种屏幕尺寸
• 支持中英文双语切换
• 多指板同时对比，每块指板可独立配置调弦、调性和音阶
• 设置自动保存在浏览器本地存储，刷新不丢失
• 支持 JSON 格式导入导出，方便备份和分享

• Runs entirely in the browser — no installation needed, all data stays on your device
• PC and mobile friendly with touch support
• Responsive design for all screen sizes
• Chinese/English bilingual support
• Multiple fretboards with independent tuning, key, and scale settings
• Settings auto-saved to browser local storage
• JSON import/export for backup and sharing
