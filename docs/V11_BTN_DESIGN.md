# 指板操作按钮 · 美术设计计划

---

## 一、当前状态（问题总结）

```
┌──────────────────────────────┐
│  指板 Canvas               ▲ │ 右上区域
│                           ▼ │
│                         ✕   │
│                             │
│  𝄞 ▼                  ⧉ [+]│ 右下 + 左下
└──────────────────────────────┘
```

### 现有问题

1. **右上角三个按钮（▲▼✕）挤在一起**，间距不统一
2. **右下角两个按钮（⧉+）间距太小**，触屏易误触
3. **左下角调音按钮（𝄞）与右下角按钮高度不一致**
4. **PC hover 与 mobile touch 体验不一致**
5. **各按钮尺寸参数分散，缺乏统一变量**

---

## 二、目标布局

```
┌──────────────────────────────┐
│                     [▲] [▼]  │  行1：上移 下移  ── 右上
│                        [✕]   │  行2：删除       ── 右上偏中
│                              │
│  [𝄞 ⏷]            [⧉] [+]  │  行3：调音  复制  新增
└──────────────────────────────┘
```

### 位置矩阵

| 按钮 | 角落 | 水平偏移 | 垂直偏移 | 宽高 |
|------|------|---------|---------|------|
| ▲ 上移 | 右上 | right: 72px | top: 4px | 28×28 |
| ▼ 下移 | 右上 | right: 40px | top: 4px | 28×28 |
| ✕ 删除 | 右上 | right: 4px | top: 36px | 28×28 |
| 𝄞 调音 | 左下 | left: 8px | bottom: 8px | 36×36 |
| ⧉ 复制 | 右下 | right: 48px | bottom: 8px | 36×36 |
| + 新增 | 右下 | right: 8px | bottom: 8px | 36×36 |

---

## 三、尺寸体系

### 按钮尺寸（直径）

| 位置 | PC 端 | 手机端 | 说明 |
|------|-------|--------|------|
| 右上角（▲▼✕） | 28px | 36px | 操作频率低，可稍小 |
| 右下角（⧉+） | 36px | 42px | 常用操作，稍大 |
| 左下角（𝄞） | 36px | 42px | 与右下角对称 |

### 间距

| 间距 | 值 | 说明 |
|------|----|------|
| ▲ 与 ▼ 之间 | 28px（中心距） | ▼ right:40px - ▲ right:72px + ▼ width:28px = 间距 4px |
| ▼ 与 ✕ 之间 | 32px（中心距） | ✕ right:4px - ▼ right:40px + ✕ width:28px = 间距 4px |
| ▲ 与 ✕ 垂直 | 32px | ✕ top:36px - ▲ top:4px - ▲ height:28px = 间隙 4px |
| ⧉ 与 + 之间 | 40px（中心距） | + right:8px - ⧉ right:48px + + width:36px = 间距 4px |
| 𝄞 与 ⧉ 之间 | 底部自然对齐 | 无水平重叠 |

### 颜色

| 按钮 | 底色（折叠） | 底色（展开/激活） | 文字色 |
|------|------------|-----------------|--------|
| ▲ | `var(--color-toggle-on)` 橙黄 | 同左 | white |
| ▼ | `var(--color-toggle-on)` 橙黄 | 同左 | white |
| ✕ | `var(--color-btn-danger)` 红 | 同左 | white |
| 𝄞 | `#4CAF50` 绿 | `#2E7D32` 深绿 | white |
| ⧉ | `#4CAF50` 绿 | 同左 | white |
| + | `#4CAF50` 绿 | 同左 | white |

## 四、显隐规则（不变）

| 平台 | 显示条件 | 隐藏条件 |
|------|---------|---------|
| PC | hover 指板任意位置 | 离开指板区域 |
| 两端（触屏/鼠标） | 激活指板时（`.show-buttons`） | 点击指板外部空白 |
| 显隐控制 | `opacity` + `pointer-events` | JS toggle `.show-buttons` |
