# 8 套调色板配色方案（v2 — 和谐版）

## 设计原则

1. **不再为功能预留特定色** — danger 只是「较深」的 accent，btn 只是「较突出」的 panel
2. **每个调色板的 10 色来自同一色系家族的亮度渐变**
3. **任何色在背景上都有 ≥ 4.5:1 对比度**
4. **C01~C04 是背景层（亮度 70~95%）、C05~C06 是文字层（亮度 8~40%）、C07~C09 是交互层（亮度 20~55%）、C10 是边框层（亮度 30~50%）**
